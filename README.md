# FinApp — AI-Powered Personal Finance Tracker

A full-stack AI-powered personal finance management web application built with the MERN stack and TypeScript. FinApp helps users track income and expenses, visualize financial data, and learn smart money habits.

---

## 🚀 Features

- 📊 **Dashboard Overview** — View available balance, total income, total expenses, and savings rate
- 💸 **Transaction Management** — Add, edit, delete, and filter transactions by category and date
- 📈 **Analytics & Charts** — Line charts for income vs expense trends, pie chart for expense breakdown
- 🤖 **AI Receipt Scanner** — Scan receipts using Google Gemini AI to auto-fill transaction details
- 📁 **CSV Import** — Bulk import transactions via CSV file
- 🔁 **Recurring Transactions** — Schedule automatic recurring transactions via cron jobs
- 📧 **Monthly Email Reports** — Automated monthly financial summary sent to your email
- 📚 **Financial Literacy** — Learn budgeting, investing, saving, tax, and debt management concepts
- 🌗 **Light/Dark Mode** — Toggle between light and dark themes
- 🔐 **JWT Authentication** — Secure login with access and refresh tokens
- 🖼️ **Profile Picture Upload** — Upload profile photos via Cloudinary
- ₹ **INR Currency Support** — Fully localized for Indian Rupee

### Backend
- Node.js + Express
- TypeScript

- Google Gemini AI
- Cloudinary
